Read Me - For Catch The Sphere - A 3D Game which is Procedurally generated using DirectX 

Folder structure -

 - Coursework Report (pdf)
 - Source Code 
	- code
	- data
	- shaders
	- Engine (contains complete Visual studio project)
 
	
Key Notes - Very Important !!!
	
	Total 3 levels are present, so after changing two levels, no more level can be changed. 
	All the textures are procedurally generated apart from the Catching Sphere texture.
	The game was developed as part of coursework requirement, so full description of the game can be found in the report.
